# avatar updater

Glitch twitter profile using [Glitch.js](http://sjhewitt.github.io/glitch.js/).

#### Install

`npm i`

#### Run

`node . && gulp`

#### Env vars

* TWITTER_CONSUMER_KEY
* TWITTER_CONSUMER_SECRET
* TWITTER_ACCESS_TOKEN
* TWITTER_ACCESS_TOKEN_SECRET

### Build

`gulp production`

### Deploy

`git push heroku master`
